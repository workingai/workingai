Apps Script

네이버 뉴스 수집기 제작

코드 생성 조건 입력

각 영역의 값만 입력하면 선택한 검색 조건에 맞는 Apps Script 코드가 자동으로 조합됩니다.

네이버 디벨로퍼 Google AI Studio

🔑 네이버 API 입력

네이버 개발자센터에서 발급한 인증 정보를 입력합니다.

API KEY

발급받은 API 키는 코드에 포함됩니다. 공유용 코드에서는 키를 별도 관리하는 것을 권장합니다.

🔎 키워드 입력

수집할 키워드 개수를 정하고, 각 검색어를 입력합니다.

KEYWORDS

각 키워드는 별도로 검색한 뒤, 결과 시트의 첫 번째 열에 어떤 키워드의 결과인지 함께 기록됩니다.

⚙️ 옵션

가져올 결과 수, 정렬 방식, 검색 대상을 설정합니다.

OPTIONS

뉴스와 블로그는 네이버 검색 API의 서로 다른 엔드포인트를 사용합니다. 선택한 대상에 맞춰 코드가 자동 구성됩니다.

구글 시트 이동

복사한 코드는 Google 스프레드시트의 확장 프로그램 → Apps Script에 붙여 넣어 사용할 수 있습니다.

✉️ 이메일 및 API 설정

이메일 수신 정보 및 Gemini API Key를 설정합니다.

SETTINGS

📝 생성 프롬프트 설정

Gemini가 생성할 뉴스 요약 및 이미지 관련 지시사항을 설정합니다.

PROMPTS
앱스 스크립트 이동

복사한 코드는 Google Apps Script의 파일 편집기 영역에 붙여 넣어 사용할 수 있습니다.
